Build a Python enrichment pipeline

Add your own extraction, routing, redaction, indexing, or AI steps.

import docframe as df

framework = df.DocFrame()

async def tag_large_docs(result):
    result.metadata.extra["large"] = result.metadata.size_bytes > 5_000_000
    return result

framework.use(tag_large_docs)
results = await framework.process_many(paths, continue_on_error=True)

Handle malformed files safely

Corpus jobs can keep going and return per-file parser errors.

results = await framework.process_many(
    ["a.pdf", "broken.pdf"],
    continue_on_error=True,
)
